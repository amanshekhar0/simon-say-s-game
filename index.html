SIMON'S GAME

Press any key to start

  • The game consists of a sequence of colored boxes that blink one at a time
  • Players must remember the order in which the boxes blinked and click on them in the same sequence.
  • Each round, the sequence increases in length, challenging players' memory and attention.
  • If a player clicks on the wrong box or clicks them in the wrong order, they lose the round.
  • The game continues until the player makes a mistake or completes the sequence..

  • Round 4: Red box blinks, then blue box blinks, then green box blinks, then yellow box blinks.
  • Player clicks on the red box, then the blue box, then the green box, then the yellow box.
